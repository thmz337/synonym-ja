# synonym-ja

## 概要

[日本語 WordNet](https://bond-lab.github.io/wnja/)を使用し、類義語を検索するコマンドです。
検索を行うと、以下の情報が出力されます。

- 概念
  その語句が持つ概念（複数の可能性あり）を示します。

- 定義
  概念が持つ意味を示します。

- 類義語
  概念に基づく類義語を示します

## 使用方法

```
npm install -g synonym-ja

synja 類義語を検索したい語句
```

## オプション

`-j`オプションをつけることで、出力をjsonにすることができます。
